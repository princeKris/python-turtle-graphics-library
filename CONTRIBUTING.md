## How to Contribute

1. Fork this repository
2. Create a new branch
3. Add your Turtle program inside the correct folder
4. Use clear filenames (example: rotating_spiral.py)
5. Commit and open a Pull Request

## Guidelines

- Use Python 3.x
- Keep code simple and beginner-friendly
- Add comments if the logic is complex
- Avoid unnecessary dependencies

## What You Can Add

- New Turtle patterns or animations
- Optical illusions
- Math art or creative visuals
- Performance or readability improvements
